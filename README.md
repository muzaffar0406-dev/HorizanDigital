## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/horizan-digital-delta.git
cd horizan-digital-delta
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
